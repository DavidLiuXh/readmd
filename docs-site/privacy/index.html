Privacy Policy

Last updated: May 2026

ReadMD does not collect, store, or transmit any personal data. All processing happens locally on your device.

What We Access

ReadMD requests the file:// host permission solely to:

  • Read Markdown files that you explicitly open or select
  • Render those files for display within the extension
  • Resolve relative image paths referenced inside Markdown files

What We Do NOT Do

  • We do not read any files you have not opened or selected
  • We do not upload, sync, or share any file contents to any server
  • We do not collect analytics, crash reports, or usage statistics
  • We do not use cookies or any form of tracking
  • We do not communicate with any remote server

Local Storage

ReadMD uses chrome.storage.local to save your UI preferences (theme, language, panel width). This data never leaves your device.

ReadMD uses IndexedDB to remember directory handles you have previously opened, so you can reopen them without picking them again. These handles are stored entirely on your device.

Permissions Explained

  • storage — saves theme, language, and panel width preferences locally
  • tabs — opens the reader page in a new tab when you click the extension icon or trigger it from a local directory page
  • host permission file:///* — enables reading local Markdown files and injecting a button into local directory pages

Third-Party Code

ReadMD bundles several open-source libraries (marked, highlight.js, KaTeX, Mermaid, Zustand) for rendering purposes. None of these libraries make network requests or collect data.

Contact

If you have any questions about this privacy policy, please open an issue on GitHub.

中文

隐私政策

最后更新：2026 年 5 月

ReadMD 不收集、不存储、不传输任何个人数据。所有处理均在您的本地设备上进行。

我们访问的内容

ReadMD 申请 file:// 主机权限，仅用于：

  • 读取您主动打开或选择的 Markdown 文件
  • 在扩展内渲染并展示这些文件
  • 解析 Markdown 文件中引用的相对路径图片

我们不做的事

  • 不读取您未主动打开的任何文件
  • 不将任何文件内容上传、同步或共享到任何服务器
  • 不收集分析数据、崩溃报告或使用统计
  • 不使用 Cookie 或任何形式的追踪
  • 不与任何远程服务器通信

本地存储

ReadMD 使用 chrome.storage.local 保存您的界面偏好（主题、语言、面板宽度），这些数据不会离开您的设备。

ReadMD 使用 IndexedDB 记录您曾经打开过的目录句柄，方便下次免选择重新打开。这些数据完全存储在您的本地设备上。

权限说明

  • storage — 在本地保存主题、语言和面板宽度偏好
  • tabs — 点击扩展图标或从本地目录页触发时，在新标签页打开阅读器
  • host permission file:///* — 允许读取本地 Markdown 文件，并在本地目录页注入操作按钮

第三方代码

ReadMD 内置了若干开源库（marked、highlight.js、KaTeX、Mermaid、Zustand）用于渲染。这些库均不发起网络请求，也不收集任何数据。

联系方式

如有隐私相关问题，请在 GitHub 提交 Issue。
